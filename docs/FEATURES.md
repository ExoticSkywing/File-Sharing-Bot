# 小芽空投机功能清单

## ✅ 已完成功能

### 1️⃣ 架构基础
- **Bot API 化改造**：Pyrogram Bot + FastAPI 内部 API（端口 18690）
- **Gateway 转发层**：统一网关接口，前端零感知 Bot 实现
- **签名认证体系**：md5 签名防篡改，内部 API 安全调用
- **Systemd 服务化**：Bot 主进程 + 内部 API 双服务，开机自启 + 崩溃重启

### 2️⃣ 资源包管理
- **创建空投包**：TG Bot 端多文件收集 → 存储频道转发 → 生成 pack_id
- **资源包列表**：分页查询、搜索过滤（名称/标签/pack_id/口令）
- **资源包详情**：查看文件列表、口令列表、分享链接
- **编辑功能**：修改资源包名称、修改标签
- **删除系统**：软删除（逻辑删除）+ 回收站 + 彻底删除（物理删除）

### 3️⃣ 口令与分享
- **口令生成**：支持自定义口令、随机口令
- **口令管理**：启用/禁用口令、设置最大使用次数、设置过期时间
- **分享链接**：生成 t.me/bot?start=xxx 格式链接
- **领取机制**：用户通过口令或链接领取资源包

### 4️⃣ 标签分类系统
- **标签打标**：为资源包添加多个标签（逗号分隔）
- **标签分组**：创建分组（如"顶级博主"、"游戏"）
- **拖拽管理**：标签拖入/拖出分组
- **智能统计**：实时统计每个标签的资源包数量
- **标签过滤**：点击标签快速筛选对应资源包

### 5️⃣ 删除与回收站
- **软删除**：资源包逻辑删除，进入回收站
- **回收站视图**：独立的回收站列表，查看已删除资源包
- **批量恢复**：从回收站批量恢复资源包
- **彻底删除**：物理删除（支持同时清理 TG 频道消息，48小时限制）
- **批量操作**：批量删除、批量恢复、批量彻底删除

### 6️⃣ 前端界面
- **响应式布局**：桌面端侧边栏 + 移动端顶部标签栏
- **卡片式展示**：资源包卡片，支持快捷操作
- **实时编辑**：标签就地编辑（点击即编辑）
- **视图切换**：正常视图 ↔ 回收站视图
- **批量模式**：多选卡片 + 批量操作工具栏

## 🚀 技术栈

**Bot 端**
- Pyrogram（TG Bot 框架）
- FastAPI（内部 API）
- MySQL（xiaoyaairdrop 数据库）
- Systemd（服务管理）

**Gateway 端**
- FastAPI（API 网关，端口 8555）
- HTTPX（内部 API 调用）
- JWT 认证

**前端**
- Vue 3 + Vite
- Element Plus
- Lucide Icons
- Vben Admin 框架
